Branch Usage Guide

How do I add more products?

Add rows to inventory_data.csv using the same column names, upload product images to the images folder, and make sure Image Filename exactly matches the uploaded image name.

How do I add more categories?

Just type the new category name in the Category column. The website automatically creates the dropdown option and category chip after refresh.

How can staff find products quickly?

Use the search box for item code, barcode, product description, flavour, colour, UOM or category. Search works across all columns.

Why is an image not showing?

The image file must be inside the images folder and the Image Filename value must match exactly, including extension such as .jpg, .png or .webp.

How do I share product details?

Use the save icon on any product card to download a clean card image for WhatsApp, email, or internal communication.
